## CSS_grammar

### 문법
* 선택자 = selector
* 속성 = property ; 
* 값 = value
* 선언 = declaration
* 선언부 = {}
* 규칙 = rule set
* 주석 = /* ~ */

### 적용방식
* inline <div style= > 내용 </div> (자주사용x)
* internal <style> ... </style>
* external <링크태그> link rel = stylesheet href='css/style.css' (주로 사용함)

### class 선택자
* css = .foo {font-size: 30px;}
* html = 태그 안에서 class 속성 지정

### 속성 선택자
* css = p[class] {color: silver;}
* html = 태그 안에서 class 속성 지정
